…or create a new repository on the command line
```bash
# 创建README文件并写入项目名
echo "# AssestManager" >> README.md
# 初始化本地Git仓库
git init
# 暂存README文件
git add .
# 提交版本并添加备注
git commit -m "first commit"
# 重命名主分支为main
git branch -M main
# 关联远程GitHub仓库
git remote add origin https://github.com/gitRebdolphin8820/AssestManager.git
# 首次推送并绑定本地与远程分支
git push -u origin main
```

…or push an existing repository from the command line

git remote add origin https://github.com/gitRebdolphin8820/AssestManager.git
git branch -M main
git push -u origin main

日常新增
```bash

git pull
# 暂存所有变更文件（. 代表当前目录所有）
git add .
# 提交版本，备注替换成实际操作（如：添加XX功能、修改配置、新增测试文件）
git commit -m "添加新文件XXX"
# 推送到GitHub远程仓库
git push
```
